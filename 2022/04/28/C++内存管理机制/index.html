C++ memory primitives

new 调用 operator new ，operator new 先调用malloc，如果调用失败，就会调用你自己设定的函数

delete:

pc->~Complex ( );//先析构

array new 和array delete

placement new
  • placement new允许我们将object建在 allocated memory 中 。
    没有所谓placement delete，因急placement new根本没分配memory.或者称placement new对应的的operator delete为 placement delete.

new delete -> operator new operator delete (局部 member)-> operator new operator delete（global 全局函数）-> malloc free(CRT functions)
