new 调用 operator new ，operator new 先调用malloc，如果调用失败，就会调用你自己设定的函数

delete:

pc->~Complex ( );//先析构
operator delete (pc) ;//然后释放内存

placement new
  • placement new允许我们将object建在 allocated memory 中 。
